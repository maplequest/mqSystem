mqColorToRGB

convert any color representation to [r,g,b]

mqColorToRGB(color: string): Array
Parameters
color (string) Any represenation of color including CSS variable
Returns
Array: color triple
Example
mqColorToRGB('red');
> [255,0,0]
mqColorToRGB('#fff');
> [255,255,255]
mqColorToRGB('var(--mqDanger)');
> [229,62,62]

mqSet

Set element properties, including attributes, styles, event handlers and methods. Multiple key, value sets can be specified in the same call, and will all be applied against the object specified in the first argument. Note that non-standard attribute keys will not work, use mqAttr for those instead. The mqSet call is central to the mqSystem concept of unifying the separate CSS/HTML/JS web development domains.

mqSet(element: (Object | String), key: String, value: Object)
Parameters
element ((Object | String)) DOM object or string identifier
key (String) String that matches attribute, style, event handler or object method
value (Object) Object of any type that corresponds to the key
Example
mqSet('myelement','color','red','background','blue','class','colorful');
mqSet('myelement','class','type1 type2');
mqSet('myelement','onclick',(e)=>{console.log('click');});
mqSet('myelement','innerHTML',"Hello World");

Parameters
obj ((Object | String)) Object or string identifier
key (String) Style name
val (String) Optional style value to set
Returns
String: The value of the style if third argument omitted.
Example
